# dP Shortcut

1. Try to represent the problem in terms of the index.
2. Do all the possible stuffs on that index according to the problem statement.
3. (a) Sum of all the stuffs --> count all the possible ways
   (b) Min(of all the stuffs) --> find min
   (c) Max(of all the stuffs) --> find max
